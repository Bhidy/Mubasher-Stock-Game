# Bhidy App - Brand Identity & Design System Report

## 1. Executive Summary
**Bhidy** represents a modern, dual-persona financial experience that bridges the gap between gamified learning ("Player Mode") and professional analytics ("Investor Mode"). The design language is built on a "Clean Modernism" philosophy, utilizing a mobile-first approach with a focus on ease of use, visual hierarchy, and premium aesthetics.

## 2. Design Philosophy
*   **Dual Nature**: The interface seamlessly adapts between two distinct modes, sharing a core structure but diverging in color psychology and interaction density.
*   **Glassmorphism & Depth**: Usage of subtle shadows (`--shadow-lg`), gradients, and blurring effects to create depth without clutter.
*   **Motion**: Micro-interactions (pulse, float, shimmer) make the app feel "alive" and responsive.

## 3. Typography System
**Primary Font**: `Outfit` (Sans-serif)
*   **Character**: Modern, geometric, friendly yet professional.
*   **Weights**:
    *   **900 (Black)**: Large Headers, Portfolio Values.
    *   **700 (Bold)**: Section Titles, Button Text.
    *   **500/400 (Medium/Regular)**: Body text, secondary info.
    *   **300 (Light)**: Subtle captions.

**Secondary Font (Arabic Support)**: `Noto Kufi Arabic`
*   Ensures legibility and modern aesthetic for localized content.

## 5. UI Component Library

### Buttons & Pills
*   **Shape**: Fully rounded (`border-radius: 999px`) for a friendly, modern feel.
*   **Style**: Gradient backgrounds with subtle drop shadows (`box-shadow: 0 4px 14px...`).
*   **Interaction**: Scale down slightly on click (`transform: scale(0.98)`).

### Cards
*   **Shape**: Large border radius (`24px` / `--radius-lg`).
*   **Style**: White background, soft shadow (`--shadow-md`), thin border for definition.
*   **Hover**: Lift effect (`transform: translateY(-2px)`) and increased shadow.

### Status Indicators
*   **Trending/Hot**: Amber background (`#fef3c7`) with orange text.
*   **Safe/Stable**: Green background (`#dcfce7`) with emerald text.
*   **Volatile**: Red background (`#fee2e2`) with red text.
